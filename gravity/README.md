# Gravity

Plataforma multi-tenant SaaS B2B modular. Uma empresa assina e acessa serviços compartilhados (email, atividades, dashboard, WhatsApp, etc.) e produtos verticais especializados — sem duplicar dados entre produtos.

## Ondas de Desenvolvimento

| Onda | O que é construído |
|:---|:---|
| Onda 1 | Esqueleto do monorepo, schemas Prisma base, Marketplace |
| Onda 2 | Núcleo UI, Shell, Configurador |
| Onda 3 | Serviços de tenant, serviços de produto, produtos |
| Onda 4 | Proxy, Auth Flow, DevOps |
